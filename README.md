# Node Hello API

This is a simple Node.js application that sets up an Express server with a single API endpoint.

## Project Structure

```
node-hello-api
├── server.js
├── package.json
└── README.md
```

## Getting Started

To run this application, you need to have Node.js and npm installed on your machine.

### Installation

1. Clone the repository or download the files.
2. Navigate to the project directory:
   ```
   cd node-hello-api
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Server

To start the server, run the following command:
```
node server.js
```

The server will start and listen on port 3000.

### Accessing the API

You can access the API endpoint by navigating to the following URL in your web browser or using a tool like Postman:
```
http://localhost:3000/hello
```

You should receive a response with a simple message.
